
# EHRI Dashboard | Environmental Health Risk Intelligence

This is a NextJS-based environmental health monitoring system.

## How to Download Your Project

To download the entire directory from Firebase Studio:

1.  **File Explorer**: Go to the explorer pane on the left side of the editor.
2.  **Right-Click**: Right-click the root folder of your project.
3.  **Download**: Select **Download** from the menu. The editor will package all files and initiate a browser download.

### Alternative (Terminal Method)
If you prefer using the terminal:
1. Open the terminal in the IDE.
2. Run: `zip -r project-backup.zip .`
3. Right-click the `project-backup.zip` file in the explorer and select **Download**.

## Project Features
- **Live Sensor Matrix**: Bento-grid visualization of PM2.5, Temperature, Humidity, and Density.
- **AI Synthesis**: AI-generated risk explanations grounded in live data.
- **Intelligence Node**: Interactive AI chatbot for deep inquiry into environmental health profiles.
- **Biomedical Impact**: Clinical-grade analysis of respiratory and cardiovascular risks.
